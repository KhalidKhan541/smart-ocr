# SmartOCR.ai

AI-powered OCR text extractor that runs in your browser. Extract text from images and PDFs instantly.

## Features
- Image & PDF text extraction
- 100+ languages support
- Copy to clipboard
- Download as TXT/DOCX
- History sidebar
- Dark mode

## Tech Stack
- React 18 + TypeScript
- Tesseract.js v5
- Tailwind CSS
- Cloudflare Pages

## Live Demo
https://smart-ocr-ai.pages.dev

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
npx wrangler pages deploy dist
```
